Tipos de selectores que entran en examen

Clase (class=””)

El atributo (class=””) permite seleccionar y acceder a elementos específicos a través de los selectores de clase
Ahora vamos a hacer un ejemplo en el que tenemos una lista con tres productos y queremos que nos identique el producto que podemos conseguir en una frutería.

  1. Manzanas
  2. carne
  3. Pantalones

Identificador (id=””)

El atributo global id define un identificador único (ID) el cual no debe repetirse en todo el documento. Su propósito es identificar el elemento al vincularlo.
Vamos a identificar el título del texto coloreandolo de rojo.

Texto descriptivo

Los textos descriptivos son aquellos que caracterizan el aspecto de un elemento, que puede ser un hecho, una persona, una situación, un objeto, un animal, etc. El texto descriptivo (que puede ser oral u escrito) caracteriza la apariencia o el aspecto de algo.

Todos los elementos (*)

Estos elemenos modifican todo el documento. Puedes modificar la letra. el margen, el tamaño etc.
Como ejemplo vamos a cambiar el margen del documento

Varios elementos ( p, em, h1, h2)

Estos elementos modifican todos los elementos del documento
Ahora todos los elemnetos p, h1 y h2 van a ser de color verde

Elementos contenidos en otros elementos (ol li)

Estos elementos modifican a los elementos incluidos dentro de otros. En esta caso los elemntos li de la lista ol cambiarán a color rosa.

Sobrevoleo (p:hover)

Identifica un elemento cuando el usuario interactúa con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor
En este ejemplo cuando pasemos el cursor por encima de la plabra apariencia, ésta aparecerá de color rosa.
